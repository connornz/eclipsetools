Eclipse Swiss Army Knife v1.0.0
================================

A comprehensive tool for managing Eclipse ESB services, Stock Exporters, 
and SQL Server components.

FEATURES:
---------
✅ Eclipse ESB Service Management (Install/Uninstall/Start/Stop/Restart)
✅ Eclipse Stock Export Service (Beta) Management
✅ Create standardized Eclipse Install directory structure
✅ Share Eclipse Install directory with network access
✅ Download Microsoft SQL Server 2022 Express
✅ Download Microsoft SQL Server Management Studio 21
✅ Configure SQL Server network protocols (Named Pipes/TCP-IP)
✅ Extract Eclipse database registry keys for backup/restore
✅ Download IIS URL Rewrite Module
✅ Download .NET 8 Core (Hosting + Desktop Bundle)
✅ Download .NET Framework 4.8
✅ Download Eclipse Update Service 1.52
✅ Download Eclipse Online Chrome 3.8.21
✅ Download Eclipse Online Server 11.40.33.0

SYSTEM REQUIREMENTS:
--------------------
• Windows 7/8/10/11
• Administrator privileges (required for service installation)
• Internet connection (for SQL Server downloads)

INSTALLATION:
-------------
1. Double-click "Eclipse Swiss Army Knife Clean.exe" (recommended)
   OR
   Double-click "Eclipse Swiss Army Knife.exe" (original version)
2. The application will automatically request Administrator privileges
3. Click "Yes" when prompted by Windows UAC
4. Follow the on-screen menu options

USAGE:
------
The tool provides an interactive menu system:
• Option 1: Eclipse ESB Service Management
• Option 2: Eclipse Stock Export (Beta) Management  
• Option 3: Create Eclipse Install Directory Structure
• Option 4: Share Eclipse Install Directory
• Option 5: Download SQL Server 2022 Express
• Option 6: Download SQL Server Management Studio 21
• Option 7: Configure SQL Server Network Protocols
• Option 8: Extract Eclipse Database Registry Keys

Eclipse Online/Aura Components:
• Option 9: Download IIS URL Rewrite Module
• Option 10: Download .NET 8 Core (Hosting + Desktop Bundle)
• Option 11: Download .NET Framework 4.8
• Option 12: Download Eclipse Update Service 1.52
• Option 13: Download Eclipse Online Chrome 3.8.21
• Option 14: Download Eclipse Online Server 11.40.33.0

NOTES:
------
• The application automatically requests Administrator privileges when needed
• SQL Server downloads require internet connection
• Registry extraction creates portable Database-Connection.reg files
• Directory sharing requires appropriate network permissions
• Eclipse Online/Aura components download to C:\Eclipse Install\Dependencies by default
• IIS URL Rewrite Module download uses updated Microsoft URL with interactive installer launch
